# 🎓 Campus Complaint Portal

A web-based system for managing campus facility complaints efficiently.
Built using **ASP.NET MVC + SQL Server + Bootstrap**.

---

## 🚀 Features

✅ Student Registration & Login
✅ Submit Complaint with Image
✅ Auto-generated Complaint ID (CFCR-101)
✅ Admin Dashboard
✅ Complaint Status Update (Pending / In Progress / Fixed)
✅ Track Complaint using Complaint ID
✅ Notifications System

---

## 🛠️ Technologies Used

* ASP.NET MVC (C#)
* SQL Server
* Entity Framework
* Bootstrap 5
* HTML, CSS, JavaScript
